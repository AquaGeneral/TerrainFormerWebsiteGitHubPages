Terrain Former Lite

Terrain Former Lite is a more affordable version of Terrain Former with the same productive core of the regular version of Terrain Former. The main difference between the full and lite versions of Terrain Former is that multi-terrain sculpting support is only present in the full version of Terrain Former.
